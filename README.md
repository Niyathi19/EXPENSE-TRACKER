A full-stack expense tracking web application built using Flask that helps users manage their personal finances efficiently. The application supports secure authentication, user-specific expense tracking, interactive data visualization, and budget monitoring, providing a real-world personal finance management experience.

 Features

User Authentication – Secure login, registration, and logout using Flask-Login

 User-Specific Data – Each user can manage their own expenses independently

 Add & Manage Expenses – Add expenses with title, amount, category, and date

 Interactive Dashboard – Visualize spending using pie and bar charts (Chart.js)

 Monthly Budget Alerts – Set a monthly budget and receive alerts when exceeded

Dark Mode Support – Toggle between light and dark themes for better UX

Analytics – Monthly & yearly expense summaries with top spending categories

 Database Management – SQLite with SQLAlchemy ORM

 Tech Stack

Backend: Flask (Python)

Frontend: HTML, Bootstrap, JavaScript

Database: SQLite

ORM: SQLAlchemy

Authentication: Flask-Login

Data Visualization: Chart.js


How to Run Locally
git clone <repository-url>
cd expense-tracker
pip install -r requirements.txt
python app.py


Then open:

http://127.0.0.1:5000/
